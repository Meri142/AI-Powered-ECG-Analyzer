# AI-Powered-ECG-Analyzer
Bu proje, **Future Talent Program - Yapay Zeka (201)** modülü bitirme projesi olarak hazırlanmıştır. Proje, EKG sinyallerini analiz ederek kalp ritmindeki anormallikleri sınıflandırmayı amaçlar.

## 🚀 Proje Hakkında
Kalp ritmi bozukluklarının (Aritmi) erken teşhisi hayati önem taşır. Bu çalışmada:
* Kaggle üzerindeki gerçek EKG veri setleri kullanılmıştır.
* Veriler görselleştirilerek analiz edilmiş ve gürültülerden arındırılmıştır.
* Makine öğrenmesi modeli ile kalp atışları 5 farklı sınıfta kategorize edilmiştir.

## 🛠️ Kullanılan Teknolojiler
* **Python** (Pandas, NumPy)
* **Veri Görselleştirme:** Matplotlib, Seaborn
* **Yapay Zeka:** Scikit-learn (veya TensorFlow/Keras)

## 📊 Nasıl Çalışır?
Sistem, ham EKG sinyallerini giriş olarak alır, bunları ön işlemeden geçirir ve eğitilmiş model sayesinde ritmin normal mi yoksa bir aritmi türü mü olduğunu tahmin eder.
